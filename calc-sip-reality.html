Why time is the risk-killer
The longer you hold, the rarer a loss.

Same market, same SIP. The only thing that changes is how long you sat still. The worst one-year run lost 36%; the worst ten-year run lost 17% and there were far fewer of them. Quitting early is the one move that locks the bad number in.
